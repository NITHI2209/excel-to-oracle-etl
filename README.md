Excel to Oracle ETL Project

Overview

This project reads student data from an Excel file using Python and loads it into an Oracle Database table.

Technologies Used

* Python
* Pandas
* Oracle Database 21c XE
* VS Code

Workflow

Excel File → Python → Oracle Database

Columns

* NAME
* FATHER_NAME
* DOB
* QUALIFICATION

Output

Successfully inserts Excel data into the Oracle STUDENT table.
